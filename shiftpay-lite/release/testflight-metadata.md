## Short Description
ClockRoster helps HR teams schedule shifts, track attendance variance, approve exceptions, and run payroll with country-aware rules.

## Long Description
ClockRoster is a lightweight HR operations app for shift-based teams.

Use ClockRoster to:
- Create and manage employees
- Build weekly schedules with templates
- Track attendance and compare scheduled vs actual hours
- Handle exceptions (OT, absence, medical, annual leave)
- Run payroll with country presets (US, AU, PH)
- Export payroll summaries

Designed for speed and clarity, ClockRoster keeps HR workflows simple while preserving audit-friendly records.
